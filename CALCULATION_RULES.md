# アグリコラリバイズドエディション 得点計算ルール

## 概要
アグリコラリバイズドエディションにおける最終得点計算の詳細ルールをまとめた文書です。
ゲーム終了時、以下の各項目について得点を計算し、その合計が最終得点となります。

## 得点計算表

### 1. 農場の開発状況

#### 1.1 畑（Grain Fields）
| 畑の枚数 | 得点 |
|---------|------|
| 0-1枚   | -1点 |
| 2枚     | 1点  |
| 3枚     | 2点  |
| 4枚     | 3点  |
| 5枚以上 | 4点  |

#### 1.2 牧場（Pastures）
| 牧場の枚数 | 得点 |
|-----------|------|
| 0枚       | -1点 |
| 1枚       | 1点  |
| 2枚       | 2点  |
| 3枚       | 3点  |
| 4枚以上   | 4点  |

#### 1.3 未使用スペース（Unused Spaces）
| 項目 | 得点 |
|------|------|
| 未使用スペース1つにつき | -1点 |

*注：未使用スペースとは、畑でも牧場でも部屋でもない農場ボード上の空きスペースを指す*

#### 4.2 部屋の材質

| 項目 | 得点 |

### 5. その他のボーナス

#### 5.1 柵で囲まれた厩（Fenced Stables）
| 項目 | 得点 |
|------|------|
| 柵で囲まれた厩1つにつき | 1点 |

*注：最大4つ（農場ボードの制限による）*

#### 5.2 カードボーナス点（Card Points）
進歩カードや職業カードに記載されたボーナス点をそのまま加算

### 6. 物乞いカード（Begging Cards）
| 項目 | 得点 |
|------|------|
| 物乞いカード1枚につき | -3点 |

## 得点計算の例

### 例1：標準的なプレイヤー
```
畑：3枚 → 2点
牧場：2枚 → 2点
未使用スペース：1つ → -1点
羊：4頭 → 2点
猪：2頭 → 1点
牛：1頭 → 1点
穀物：5個 → 2点
野菜：2個 → 2点
家族：4人 → 12点（4×3）
粘土の部屋：3つ → 3点（3×1）
石の部屋：1つ → 2点（1×2）
柵で囲まれた厩：2つ → 2点（2×1）
カードボーナス：5点
物乞いカード：0枚 → 0点
----------------------------------------
合計：35点
```

### 例2：家畜特化型プレイヤー
```
畑：2枚 → 1点
牧場：4枚 → 4点
未使用スペース：0 → 0点
羊：8頭 → 4点
猪：7頭 → 4点
牛：6頭 → 4点
穀物：1個 → 1点
野菜：0個 → -1点
家族：3人 → 9点（3×3）
粘土の部屋：2つ → 2点（2×1）
石の部屋：0 → 0点
柵で囲まれた厩：4つ → 4点（4×1）
カードボーナス：3点
物乞いカード：1枚 → -3点
----------------------------------------
合計：32点
```

## 重要な注意事項

### 得点計算時の確認ポイント

1. **マイナス点の項目**
   - 各カテゴリー（畑、牧場、家畜、作物）で0または少ない場合は-1点
   - 未使用スペース1つにつき-1点
   - 物乞いカード1枚につき-3点

2. **上限値**
   - 各カテゴリーの最高点は4点（家族と部屋を除く）
   - 家族は最大5人（15点）
   - 柵で囲まれた厩は最大4つ（4点）

3. **よくある間違い**
   - 木の部屋を得点に含めてしまう（木の部屋は0点）
   - 畑や牧場の上にある作物や家畜を二重にカウントしてしまう
   - 家の中にいる家畜をカウントし忘れる
   - 収穫時に取った作物と最終的に残った作物を混同する

### リバイズドエディション特有のルール

1. **基本ゲームとの主な違い**
   - 得点計算表が簡略化されている
   - 一部のカテゴリーで得点の閾値が調整されている
   - 進歩カードと職業カードの種類と効果が異なる

2. **拡張版使用時の注意**
   - 拡張版のカードを使用する場合、カードに記載された特殊な得点ルールに従う
   - 一部の拡張カードは基本の得点計算を変更する場合がある

## 得点記録シート項目

効率的な得点計算のため、以下の順序で記録することを推奨：

1. 農場要素（畑、牧場、未使用スペース）
2. 家畜（羊、猪、牛）
3. 作物（穀物、野菜）
4. 家族数
5. 部屋（粘土、石）
6. 柵で囲まれた厩
7. カードボーナス
8. 物乞いカード（マイナス点）
9. 合計点
